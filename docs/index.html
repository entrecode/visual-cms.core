API

This is the API documentation of the Visual CMS module.

vcms

Visual CMS Core module

vcms
Example
const vcms = require('visual-cms.core');

vcms.register

Register a new Element Class for usage with Visual CMS.

vcms.register(elementsToAdd: ...Class)
Parameters
elementsToAdd (...Class) Subclasses of Element to register

vcms.getClass

Return the correct class for a type identifier string

vcms.getClass(type: string): Class
Parameters
type (string) The type of the element to get, e.g. paragraph ;
Returns
Class: the Class
Throws
  • any: Error when the class is not found / not registered yet.

vcms.parse

Parse a Visual-CMS JSON into Visual-CMS Objects

vcms.parse(json: JSON): (Element | Array<Element>)
Parameters
json (JSON) a valid JSON to parse
Returns
(Element | Array<Element>): Element or List of Elements
Throws
  • any: Error when unknown types are used.

vcms.elements

elements contains all predefined Classes, e.g. elements.Element

vcms.elements

vcms.elements.Element

Classes

The logic of Visual CMS is based around Classes with heavy inheritance. The base class, Element, contains most of the logic. The other Elements are all subclasses of Element and overwrite stuff where needed.

Link

A HTML <a> tag. Subclass of PhrasingElement. Can contain PhrasingElements and Text. href, newTab and rel are settings options.

new Link()

Extends PhrasingElement

Example
new Link({
  settings: {
    href: 'https://example.com',
    newTab: true,
    rel: ['nofollow'],
  },
  content: 'Link Text'
});

Resulting HTML:

<a href="https://example.com" target="_blank" rel="nofollow">Link Text</a>

Image

A HTML <img> tag. Subclass of FlowElement. Has no content. Depending on the properties, it is wrapped in an <a> tag or even an <div>. It also has settings options for responsive image variants using src-set.

new Image()

Extends FlowElement

Example

Simple Image:

new Image({
  settings: {
    src: 'https://example.com/image.jpg',
    alt: 'The alt attribute is always required',
    title: 'The title attribute is inferred from BaseElement',
    width: 100, // width and height are optional
    height: 200,
  }
});

Resulting HTML (alt and title values omitted for brevity):

<img src="https://example.com/image.jpg" alt="…" title="…" width="100" height="200">

Responsive Image:

new Image({
  settings: {
    src: 'https://entreco.de/image.png',
    alt: 'simple image',
    class: ['awesome'],
    title: 'whaaaat',
    responsive: {
      srcs: {
        '500w': 'https://entreco.de/image-500.png',
        '800w': 'https://entreco.de/image-800.png',
        '2x': 'https://entreco.de/image-2x.png',
      },
      sizes: {
        default: '500w',
        '(min-width: 400px)': '800w',
      },
    },
  },
});

Resulting HTML:

<img src="https://entreco.de/image.png" alt="simple image"
srcset="https://entreco.de/image-500.png 500w, https://entreco.de/image-800.png 800w, https://entreco.de/image-2x.png 2x"
sizes="(min-width: 400px) 800w, 500w"
title="whaaaat" class="awesome">

Image with Link:

new Image({
  settings: {
    src: 'https://example.com/image.jpg',
    alt: 'The alt attribute is always required',
    title: 'The title attribute is inferred from BaseElement',
    href: 'https://entrecode.de',
    rel: ['nofollow'],
    newTab: true,
  }
});

Resulting HTML (alt and title values omitted for brevity):

<a href="https://entrecode.de" target="_blank" rel="nofollow">
  <img src="https://example.com/image.jpg" alt="…" title="…">
</a>

Wrapped Image:

new Image({
  settings: {
    src: 'https://example.com/image.jpg',
    alt: 'description',
    class: ['awesome'],
    wrap: true,
  }
});

Resulting HTML (note that the class attribute is rendered on the outermost element):

<div class="awesome"><img src="https://example.com/image.jpg" alt="description"></div>

List

A HTML <ul> or <ol> tag. Subclass of FlowElement. Can contain PhrasingElements and Text. Has the settings property ordered that defines if <ul> or <ol>.

new List()

Extends FlowElement

Example
new List({
  settings: {
    ordered: false,
  },
  content: [
    new ListElement({content: 'Text'}),
  ],
});

Resulting HTML:

<ul>
  <li>Text</li>
</ul>

Block

The simplest possible Module. Simply a <div> containing FlowElements.

new Block()

Extends Module

Example
new Block({
  content: [
    new Headline({level: 1, content: 'Hello World!'}),
    new Paragraph({content: 'Text'}),
  ],
});

Resulting HTML:

<div>
  <h1>Hello World!</h1>
  <p>Text</p>
</div>
